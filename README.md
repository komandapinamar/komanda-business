# Komanda Business

Astro-powered public acquisition website for Komanda at `komanda.com`.

## Scope

- Marketing and SEO content.
- Commercial plan presentation.
- Business-registration experience.
- Acquisition analytics.

Operational accounts, tenants, memberships, catalog, checkout, orders, payments, and
printing belong to Komanda Core. This project consumes versioned Core contracts and
never accesses the Core database directly.

See [the project constitution](.specify/memory/constitution.md) for ownership,
static-first delivery, contract, performance, and release rules.
